# mindful entry

i made this extension because i was doomscrolling way too much on x and linkedin. i just needed a second to think "why am i actually opening this?" before diving in.

it blocks the feed until you type your intention. once you do, you get 5 minutes of access.

### how to install
1. download/save these files into a folder.
2. go to `chrome://extensions` in your browser.
3. toggle **developer mode** on (top right).
4. click **load unpacked** and select the folder.
